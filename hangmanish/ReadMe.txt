Hangman

A word is selected at random for player to guess.
User will select and enter a letter.
If the letter is in the word the computer has selected, it will display in the word.
If the letter is not in the word, an attempt has been used.
User will begin with 9 attempts to complete the word.

For better reference, the letters previously selected by player will be displayed along with current state of completion of the word being guessed.
